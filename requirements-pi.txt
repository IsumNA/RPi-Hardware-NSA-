# Raspberry Pi CSI camera — pip-only (no sudo apt)
#
# Install into your venv on the Pi:
#     source .venv/bin/activate
#     pip install -r requirements-pi.txt
#
# These are also pulled automatically on aarch64 Linux when you run:
#     pip install -r requirements.txt
#
# WHAT PIP CAN INSTALL (this file):
#   picamera2      — Python API for the CSI camera
#   rpi-libcamera  — libcamera Python bindings
#
# WHAT PIP CANNOT INSTALL (needs lab admin + sudo apt):
#   rpicam-hello, rpicam-vid, libcamera-hello, libcamera-vid
#   → package: libcamera-apps
#   GStreamer libcamerasrc plugin → gstreamer1.0-libcamera
#
# On a non-Pi machine (AI server, x86 laptop) this file will not help — there is
# no CSI camera. Use:  python live.py --source sim  or a USB webcam on Windows.
#
# After install:  python pi_camera_check.py

picamera2>=0.3.0
rpi-libcamera>=0.1.0
